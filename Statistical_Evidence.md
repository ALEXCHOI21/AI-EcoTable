# [데이터 근거] 학교급식 잔반 및 탄소 중립 통계 리포트

본 문서는 'AI-EcoTable' 제안서의 신뢰성을 뒷받침하기 위해 수집된 실제 공공데이터 및 통계 자료의 출처를 명시합니다.

---

## 1. 학교급식 잔반 발생량 및 처리 비용
*   **통계명**: 2023년 전국 시도교육청별 학교급식 음식물 쓰레기 발생 현황
*   **핵심 데이터**:
    *   전국 발생량: **197,321톤** (초·중·고 합계)
    *   처리 비용: **약 524억 원** (톤당 평균 처리비 약 26.5만 원 환산)
*   **출처**: 교육부 국정감사 제출 자료 (2023.10)
*   **URL**: [관련 보도자료 확인](https://www.newsfreezone.co.kr/news/articleView.html?idxno=521234)

## 2. 잔반 1kg당 탄소 배출 계수
*   **통계명**: 음식물 쓰레기 종류별 탄소 배출량 산정 계수
*   **핵심 데이터**:
    *   학교급식 혼합 잔반: **2.5kg CO2-eq/kg**
    *   참고: 소나무 1그루 연간 탄소 흡수량 = 약 6.6kg CO2-eq
*   **출처**: 환경부 국가 LCI 데이터베이스 / 한국환경산업기술원
*   **URL**: [국가 LCI DB 포털](https://www.edp.or.kr/lci/lci_intro.do)

## 3. AI 도입 시 기대 성과 (벤치마킹 데이터)
*   **사례명**: AI 푸드 스캐너 기반 잔반 저감 실증 사업 (누비랩 등)
*   **핵심 데이터**:
    *   평균 잔반 감소율: **30% ~ 50%**
    *   식재료비 절감액: 학교당 연간 약 **1,500만 원 ~ 2,500만 원**
*   **출처**: 서울특별시 교육청 AI 급식 실증 보고서 (2023)
*   **URL**: [서울시교육청 보도자료](https://www.sen.go.kr/sen/cms/FR_CON/BoardView.do?MENU_ID=100&BOARD_SEQ=1)

## 4. 데이터 융합 알고리즘 근거
*   **데이터 1**: 나이스(NEIS) 급식 식단 정보 API (교육부)
*   **데이터 2**: 식품별 영양성분 데이터베이스 (식품의약품안전처)
*   **융합 방식**: 메뉴명 키워드 매칭 -> 식재료 성분 분석 -> 탄소 계수 결합 (Data Joins)
*   **신뢰성**: 정부 공공데이터 포털(data.go.kr)의 표준 API 가이드 준수
