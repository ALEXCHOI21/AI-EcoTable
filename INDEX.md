# 📑 AI-EcoTable 프로젝트 인덱스

## 📋 프로젝트 개요
**제목**: AI-EcoTable: 공공데이터와 AI를 활용한 학교 급식 잔반 예측 및 탄소 배출 저감 솔루션

**목표**: 전국 12,000개 학교의 음식물 쓰레기를 85% 감축하고, 연간 30.9톤(30,942kg CO2-eq)의 탄소 배출을 저감

**추진 기간**: 1년 실증 (Q1~Q4)

---

## 📊 당위성 기반 데이터

### 1. 경제적 손실
- **연간 급식 잔반 처리 비용**: 524억 원 (2023년 기준)
  - 서울시: 연 68억 원 (2022년 기준)
  - 3년간(2020~2022) 서울 학교급식 잔반처리비 총 146억 원
- **학교별 월평균 처리비**: 약 63만 원 (800명 규모 학교 기준)
- **증가 추세**: 급식 인원 감소 중에도 처리비 증가

### 2. 환경 오염 문제
- **일일 발생량**: 14,000톤 (전체 쓰레기의 28.7%)
- **연간 온실가스**: 885만 톤 CO2e
- **1인당 배출**: 95kg (UNEP 기준 과다)
- **탄소 감축 효과**: 음식물 쓰레기 20% 감소 = 온실가스 177만 톤 감소 (승용차 47만 대 배출량 상당)
- **재활용 문제**: 재활용률 98%에도 불구하고 퇴비화 과정에서 메탄 54% 발생

### 3. 학교 급식 특수성
- **명절 기간 증가**: 설날/추석 기간 평소 대비 20% 이상 증가
- **하드웨어 의존성**: 기존 AI 스캐너 도입비 수백만 원 → 전국 확산 불가능
- **예산 낭비**: 음식 조리 → 버려짐 → 처리 → 탄소 배출의 악순환

## 💡 AI 기반 해결 사례

### 울주군 천상고등학교 사례 (AI 푸드 스캐너)
- **성과**: 학생당 222.3g → 163g (26.6% 감축)
- **누적 감축**: 900명 학생 2개월 2,600kg 감축
- **효과**: 실시간 모니터링으로 학생 참여도 증대

### 누비랩 AI 푸드스캔 사례
- **평균 감축율**: 30% 이상
- **최대 감축율**: 45% (일부 학교)
- **기술**: 이미지 인식 + 센서 기반 실시간 분석
- **개선점**: 데이터 축적으로 수요 예측 정확도 향상

## 🎯 AI-EcoTable의 차별성

### 기존 솔루션의 한계
1. **고가 하드웨어**: 대당 수백만 원 → 전국 도입 불가능
2. **설치 비용**: 수억 원대 인프라 투자 필요
3. **유지보수**: 고장 시 대응 지연, 운영 비용 가중

### AI-EcoTable의 강점
1. **소프트웨어 기반**: 하드웨어 의존 제거 (Zero 인프라 비용)
2. **데이터 중심**: 나이스(NEIS) API + 국가 LCI 데이터베이스 활용
3. **실시간 예측**: 영양사 PC에서 즉시 조리량 가이드 제공
4. **확산성**: 모든 학교에 즉시 적용 가능
5. **경제성**: 절감된 비용을 급식 품질 향상에 재투자
