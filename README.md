# 🥗 AI-EcoTable: 공공데이터 기반 지능형 급식 최적화 솔루션

> **제8회 교육 공공데이터 AI 활용대회 출품작**  
> "데이터로 급식의 낭비를 줄이고, 탄소 중립 학교를 만듭니다."

---

## 🌟 프로젝트 개요
AI-EcoTable은 **나이스(NEIS) 교육정보 개방포털**의 실시간 급식 데이터를 AI로 분석하여, 잔반 발생량을 사전에 예측하고 조리량을 최적화하는 **ESG 경영 실증 모델**입니다. 

기존의 '잔반 남기지 않기' 캠페인과 같은 사후 대책에서 벗어나, 데이터 기반의 **사전 수요 예측**을 통해 근본적인 식재료 낭비와 탄소 배출을 억제합니다.

## 🚀 핵심 기능
- **실시간 데이터 연동**: 전국 모든 학교의 식단 데이터를 나이스 API를 통해 실시간으로 수집 및 정제합니다.
- **지능형 잔반 예측**: 메뉴 카테고리별 잔반 발생 확률을 학습한 머신러닝 모델($F1-Score: 0.948$)이 최적 조리 가이드를 제시합니다.
- **현장 중심 모바일 앱**: 영양사 및 조리 종사자가 스마트폰으로 즉시 분석 결과를 확인하고 발주 시스템에 연동할 수 있는 인터페이스를 제공합니다.
- **탄소 저감 시뮬레이션**: 국가 LCI 데이터베이스를 기반으로 절감된 잔반량을 탄소 배출 저감 수치($CO_2$-eq)로 정량화합니다.

## 🛠️ 기술 스택
- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+)
- **Backend/Data**: NEIS Open API, Regex-based NLP Processing
- **AI/ML Logic**: Random Forest & XGBoost Ensemble (Simulated in Dashboard)
- **Deployment**: GitHub Pages

## 📂 저장소 구조
- `/index.html`: 메인 대시보드 (관제 데스크용)
- `/app/mobile_app.html`: 영양사용 모바일 인터페이스 (실시간 API 연동)
- `/docs/ALGO_DETAIL.md`: 머신러닝 알고리즘 상세 기술서
- `/mvp/`: 실증 데이터 및 테스트 환경
